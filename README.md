# NixTraceability 🚀

**NixTraceability** is a professional-grade industrial scanning and traceability system designed for modern manufacturing environments. Built with .NET 10 and WPF, it provides a robust solution for tracking parts, managing configurations, and generating production reports.

## ✨ Features

- **Industrial Scanning Interface**: Optimized for high-speed scanning with real-time validation.
- **Traceability Management**: Track part history, timestamps, and operator details.
- **SQLite Integration**: Local database support for reliable data storage without complex server setups.
- **Customizable Reports**: Generate and export reports based on date ranges and status.
- **Premium UI/UX**: Modern dark-themed interface built using Guna.UI2 and Siticone Desktop components.
- **Validation Logic**: Intelligent duplicate check and part-specific validation protocols.

## 🛠️ Tech Stack

- **Framework**: .NET 10.0-windows (WPF)
- **Database**: System.Data.SQLite
- **UI Libraries**: 
  - Guna.UI2.WinForms
  - Siticone.Desktop.UI
- **Data Handling**: Newtonsoft.Json

## 🚀 Getting Started

### Prerequisites
- Visual Studio 2022 or later
- .NET 10 SDK

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ownasd/NixTraceability.git
   ```
2. Open `NixTraceability.sln` in Visual Studio.
3. Restore NuGet packages.
4. Build and Run the project.

## 📊 Project Structure

- `NixTraceability/`: Main application source code.
- `NixTraceability_Setup.iss`: Inno Setup script for creating installers.
- `publish_app.ps1`: Automation script for publishing builds.
